# BetterPMMP
마인크래프트 **베드락 에디션** 서버를 여는 프로그램인 **PocketMine-MP**의 유지보수 후속작입니다. 성능, 게임 플레이, 관리 편의를 높이는 개선이 프로그램 안에 이미 반영되어 있습니다.

> 🌏 [English README](README.md)

## 소개

마인크래프트 베드락 서버를 열려면 서버 프로그램이 필요하며, 대표적인 것이 [PocketMine-MP](https://github.com/pmmp/PocketMine-MP)입니다. 그러나 이 프로그램은 2026년 7월 9일 개발이 종료되어 더 이상 새 버전이 나오지 않습니다.

BetterPMMP는 그 프로그램을 이어받아 유지보수를 계속하는 후속 프로젝트입니다. PocketMine-MP 5.44.3 버전을 기반으로 하며, 서버 구동에 필요한 파일이 [source/](source/) 폴더에 모두 포함되어 있습니다. 따라서 별도의 설치 과정 없이 내려받아 바로 실행할 수 있습니다.

BetterPMMP는 마인크래프트 제작사 Mojang 및 원 PocketMine 팀과 무관합니다. 원저작자 표시와 변경 내역은 [NOTICE](NOTICE)를 참고하십시오.

## 주요 기능

서버를 가볍게 만드는 성능 개선은 처음부터 켜져 있으며, 게임 플레이는 원래 그대로입니다. 게임 동작이 달라지는 기능(밝기 고정, 월드별 설정, PvP용 절약 옵션 등)만 처음엔 꺼져 있고, 필요할 때 설정 파일(`pocketmine.yml`)에서 직접 켜면 됩니다.

### 서버 관리

- **게임 내 재시작**: 채팅창에 `/restart`를 입력하면 서버를 완전히 껐다 켠 것처럼 깨끗하게 재시작합니다.
- **수정 내용 즉시 반영**: 서버 코드를 수정하면 다음 실행 시 곧바로 적용됩니다. 별도의 빌드 과정이 없습니다.
- **정돈된 시작 화면**: 서버 시작 시 출력되는 메시지를 정리하고, 기록/오류 파일의 저장 위치를 한곳으로 통합했습니다.

### 성능

- **블록 설치/파괴 렉 수정**: 블록을 놓거나 부술 때 잠시 튕겼다가 원위치로 돌아오는 현상(고무줄 현상)을 제거했습니다.
- **밝기 계산 생략**: 월드의 밝기를 매번 다시 계산하지 않고 고정하여 서버 부하를 크게 줄입니다. 밝기가 균일해지므로 필요할 때만 설정에서 켜십시오.
- **월드별 시야 거리**: 월드마다 보이는 거리를 다르게 지정할 수 있습니다.
- **월드별 자연 변화 조절**: 작물 성장, 잔디 번짐, 나뭇잎 마름처럼 저절로 일어나는 변화가 적용되는 범위를 월드별로 지정하거나 완전히 끌 수 있습니다.
- **내부 처리/통신 최적화**: 서버 내부 동작과 데이터 송수신 방식을 여러 부분 개선하여 전체 부하를 낮췄습니다.
- **PvP 서버용 절약 옵션**: 대전(아레나) 서버에 거의 필요 없는 기능(경험치 오브, 폭발에 의한 블록 파괴, 바닥 아이템 병합 등)을 꺼서 서버를 가볍게 만드는 선택 옵션입니다.

### 게임 플레이

- **크리티컬(치명타) 설정**: 치명타 동작을 설정에서 조정할 수 있습니다. 기본값은 원래 게임과 동일합니다.
- **철문 손 조작 차단**: 손으로 철문을 여닫을 수 없게 합니다. 버그 모션이 제거되었습니다.

## 요구 사항

- **Windows** (시작 스크립트가 `start.cmd` 기준)
- **PHP**: 서버 프로그램 구동에 필요한 실행 도구입니다. `source/bin/php/php.exe`에 포함되어 있어 별도 설치가 필요 없습니다.

## 설치

1. 이 저장소를 내려받습니다(초록색 **Code** 버튼 → **Download ZIP**, 또는 `git clone`).
2. `start.cmd`를 실행하여 서버를 시작합니다.

이게 전부입니다. 위에서 소개한 개선 사항이 이미 모두 반영되어 있습니다.
